# number-prettier 
Pretty a number and shortened it to make it look more pretty and simple.

## How To Use ‚ùì
```js
const { format } = require('number-prettier')
format(1000, 2) // 1k
```
## Available Values
```json
{
"k": "thousands",
"M": "millions",
"B": "billions",
"T": "trillions",
"Q": "quadrillions",
"QT": "quintillions"
}
```
